# CodeAlpha_LanguageTranslator

## 🌐 Language Translation Tool

A simple web-based language translation tool built using **Python** and **Streamlit**. Users can enter text, select a source and target language, and get an instant translation. Includes an optional text-to-speech feature to listen to the translated text.

### 🚀 Features
- Enter any text to translate
- Select source language (or auto-detect)
- Select target language from a wide list of supported languages
- Instant translation using Google Translate (via `deep-translator`)
- Optional: Listen to the translated text using text-to-speech (`gTTS`)

### 🛠️ Tech Stack
- Python
- Streamlit (UI)
- deep-translator (translation API)
- gTTS (text-to-speech)
